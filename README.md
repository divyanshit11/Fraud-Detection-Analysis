# Fraud Detection Analysis

A machine learning project focused on detecting fraudulent transactions using classification algorithms and financial transaction data.

## What it does

* Cleans and preprocesses transaction data
* Performs exploratory data analysis and feature engineering
* Encodes categorical variables for machine learning
* Trains multiple classification models including Logistic Regression, Random Forest, and XGBoost
* Evaluates model performance using training and validation accuracy metrics

## Tech Stack

* Python
* Pandas, NumPy
* Matplotlib, Seaborn
* Scikit-learn
* XGBoost

## Conclusion

* Data preprocessing and feature engineering improved fraud detection performance
* Comparing multiple classification models helped identify effective fraud prediction approaches
* Machine learning techniques successfully detected suspicious transaction patterns
